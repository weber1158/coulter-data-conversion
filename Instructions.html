Converting Beckman Multisizer4 files to .csv

Step 1. Save everything on a flash drive

Make copies of each .#m4.XLS file as well as a copy of XLStoCSV.ps1 and save everything on a flash drive.

Step 2. Move the files to your personal computer

Ideally, your personal computer runs on Windows 10 with PowerShell 5.1 or later installed.

Move the files from the flash drive onto your personal computer, and make sure they are all in the same folder.

You will need to copy the path of the folder. For example:

C:\JoeShmith\Data\Folder\

Step 3. Open PowerShell

Use the keyboard shortcut ⊞ Win + R to open the Run app.

In the Run app, type powershell_ise and hit enter.

This will launch the PowerShell Integrated Scripting Environment (ISE).

Step 3. Navigate to the correct folder

In the PowerShell ISE, type:

Set-Location "<path>"

where <path> is the name of the file path that you copied in Step 2.

For example:

Set-Location "C:\Users\JoeSmith\Data\Folder\"

Once you are finished, press Enter. This will change the the PowerShell directory to the appropriate folder.

Step 4. Execute the XLStoCSV.ps1 function

In the PowerShell ISE console, execute the following command:

PowerShell -ExecutionPolicy Bypass -File .\XLStoCSV.ps1

You need to press Enter when you're finished, otherwise the command won't execute. It may take a while for the script to run, depending on how many files are in the folder, but it will tell you when it is completed.

The final results will be saved in a new folder called CSV_Files. In this folder, there will be a file called all_data.csv that contains all of the Coulter Counter data in a single spreadsheet.
